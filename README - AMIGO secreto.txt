# 🎁 Projeto Amigo Secreto

Este é um projeto simples de **Amigo Secreto**, desenvolvido em **JavaScript, HTML e CSS**. O objetivo é permitir que os usuários adicionem participantes e realizem um sorteio aleatório para definir quem será o amigo secreto de cada um.

Este projeto faz parte do **Challenge Amigo Secreto** do curso **Praticando Lógica de Programação**, da **Alura Latam** e da **Oracle Next Education**, ano **2025**.

---

## 🚀 Funcionalidades

✅ **Adicionar participantes**: Permite que os usuários adicionem amigos à lista de participantes.

✅ **Exibir participantes na tela**: Mostra, em tempo real, os nomes adicionados.

✅ **Validar entradas**: Garante que o usuário não adicione nomes vazios ou apenas espaços.

✅ **Sortear um amigo secreto**: Escolhe aleatoriamente um nome da lista e exibe o resultado na tela.

✅ **Limpar o campo de entrada**: Após adicionar um participante, o campo de input é limpo automaticamente.

✅ **Prevenção de sorteio inválido**: Impede o sorteio se houver menos de dois participantes.

## 🎯 Como Usar

1️⃣ **Clone o repositório**
```bash
git clone https://github.com/seu-usuario/amigo-secreto.git
```

2️⃣ **Abra o arquivo** `index.html` em um navegador.

3️⃣ **Adicione os participantes** digitando os nomes e clicando no botão de adicionar.

4️⃣ **Clique no botão "Sortear"** para descobrir o amigo secreto escolhido aleatoriamente.
